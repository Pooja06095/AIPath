🔍 SELECT Queries

Learn how to retrieve data from MySQL tables using different SELECT statements.

Lesson 5 of 10

🔍 What is SELECT?

The SELECT statement is used to retrieve data from one or more tables in a MySQL database.

It is one of the most frequently used SQL commands because it allows users to view stored information.

💡 Why Use SELECT?

👀 View Records
🔍 Search Data
📊 Analyze Information
📋 Generate Reports

💻 Basic Syntax

SELECT * FROM table_name;

Replace table_name with the name of the table from which you want to retrieve data.

📋 Example Table

Students
            
            +----+---------+-----+-----------+
            | ID | Name    | Age | Course    |
            +----+---------+-----+-----------+
            | 1  | Pooja   | 20  | CSE       |
            | 2  | Rahul   | 21  | IT        |
            | 3  | Aman    | 22  | AIML      |
            | 4  | Neha    | 20  | CSE       |
            +----+---------+-----+-----------+

We will use this Students table to understand different SELECT queries.

⭐ SELECT *

SELECT * FROM Students;

The * symbol means "all columns". This query displays every record and every column from the Students table.

📄 Select Specific Columns

SELECT Name, Course
            FROM Students;

This query displays only the Name and Course columns instead of showing the complete table.

🎯 WHERE Clause

SELECT *
            FROM Students
            WHERE Course = 'CSE';

The WHERE clause filters records and returns only the rows that match the given condition.

📊 ORDER BY

SELECT *
            FROM Students
            ORDER BY Age;

ORDER BY sorts the records in ascending order by default. You can also use DESC to sort in descending order.

🔢 LIMIT

SELECT *
            FROM Students
            LIMIT 2;

The LIMIT clause restricts the number of rows returned by the query. Here, only the first two records are displayed.

🌍 Real World Example

Imagine you are working in a college management system.

👨‍🎓 View All Students
🎯 Find CSE Students
📊 Sort by Age
🔍 Display Top 5 Records

All these tasks are performed using different SELECT queries.

💬 Common Interview Question

Question

What is the difference between SELECT * and selecting specific columns?

Answer

SELECT * retrieves all columns, whereas selecting specific columns returns only the required data, making the query faster and more efficient.

💡 Did You Know?

Professional developers avoid using SELECT * in large projects because retrieving only the required columns improves performance.

📝 Key Points

  • ✅ SELECT retrieves data from tables.
  • ✅ SELECT * displays all columns.
  • ✅ WHERE filters records.
  • ✅ ORDER BY sorts the result.
  • ✅ LIMIT restricts the number of rows.
