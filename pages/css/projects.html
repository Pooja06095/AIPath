Congratulations! Now it's time to apply everything you've learned by building small CSS projects. Practice is the key to becoming a great web designer.

🚀 Why Build Projects?

Building projects helps you improve your CSS skills. It teaches you how to combine layouts, colors, Flexbox, Grid and Responsive Design into real websites.
