🎯 What are CSS Selectors?

CSS Selectors are used to select HTML elements so that styles can be applied to them. They help target specific elements on a webpage.

📚 Types of CSS Selectors

  • Element Selector - Selects HTML tags.
  • Class Selector (.class) - Selects elements with a class.
  • ID Selector (#id) - Selects one unique element.
  • Universal Selector (*) - Selects all elements.
  • Group Selector (,) - Selects multiple elements.

💻 Example


        h1{
            color:blue;
        }
        
        .red{
            color:red;
        }
        
        #title{
            font-size:30px;
        }
