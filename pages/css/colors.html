🎨 Colors & Backgrounds

Course Progress

CSS Colors and Backgrounds help make webpages attractive and visually appealing. They allow you to style text, backgrounds, and create modern designs.

🎨 What are CSS Colors?

CSS provides different ways to add colors to text, backgrounds, borders, and many other elements. Colors improve readability and user experience.

🌈 Ways to Define Colors

  • Color Name → red, blue, green
  • HEX → #ff0000
  • RGB → rgb(255,0,0)
  • RGBA → rgba(255,0,0,0.5)
  • HSL → hsl(120,100%,50%)
