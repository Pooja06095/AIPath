📦 CSS Box Model

Course Progress

The CSS Box Model explains how every HTML element is treated as a rectangular box. It consists of Content, Padding, Border, and Margin.

📦 What is the CSS Box Model?

Every HTML element is considered a box. The Box Model helps us understand the space occupied by an element and how padding, border, and margin work together.

🧩 Parts of the Box Model

  • Content → The actual text or image.
  • Padding → Space inside the border.
  • Border → Surrounds the content and padding.
  • Margin → Space outside the border.

📊 Box Model Structure

        Margin
        ┌─────────────────────────────┐
        │          Border             │
        │  ┌───────────────────────┐  │
        │  │       Padding         │  │
        │  │  ┌───────────────┐    │  │
        │  │  │    Content    │    │  │
        │  │  └───────────────┘    │  │
        │  └───────────────────────┘  │
        └─────────────────────────────┘

💻 Example


        .box{
            width:200px;
            padding:20px;
            border:3px solid blue;
            margin:30px;
            background:#f3f4f6;
        }
