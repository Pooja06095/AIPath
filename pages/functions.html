Lesson Progress

Functions help us organize reusable blocks of code in Python.

What are Functions?

A function is a reusable block of code that performs a specific task. It helps make programs shorter, cleaner and easier to maintain.

Example


        def greet():
        print("Welcome to AIPath")
        
        greet()

Output


        Welcome to AIPath

💡 Key Points

  • Functions avoid code repetition.
  • Functions are created using the def keyword.
  • A function can accept parameters.
  • A function can return values using return.
