🔁 Python Loops

Lesson 5 of 9

Loops are used to execute a block of code repeatedly until a condition is met.

What are Loops?

Loops help us repeat the same block of code multiple times without writing it again and again.

Example


        for i in range(1, 6):
        print(i)

Output


        1
        2
        3
        4
        5

💡 Key Points

  • Loops are used to repeat a block of code.
  • Python mainly provides for and while loops.
  • range() is commonly used with for loops.
  • Proper indentation is required inside loops.
