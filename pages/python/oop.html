Object Oriented Programming (OOP) helps organize code using classes and objects.

What is Object Oriented Programming?

Object Oriented Programming (OOP) is a programming approach that uses classes and objects to organize code efficiently.

Example


        class Student:
        
        def __init__(self, name):
        self.name = name
        
        student = Student("Pooja")
        
        print(student.name)

Output


        Pooja

💡 Key Points

  • OOP is based on classes and objects.
  • A class is a blueprint for creating objects.
  • An object is an instance of a class.
  • OOP improves code reusability and organization.

📝 Practice

Create a class named Car and print its brand name. 


        class Car:
        
            brand = "Toyota"
        
        print(Car.brand)
