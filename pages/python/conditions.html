🔀 Conditional Statements

Lesson 4 of 9

Conditional statements help us make decisions in Python.

What are Conditional Statements?

Conditional statements allow a program to make decisions based on different conditions.

Example


        age = 18
        
        if age >= 18:
            print("Eligible to Vote")
        else:
        print("Not Eligible")

Output


        Eligible to Vote

💡 Key Points

    • Use if to check a condition.
    • Use else when the condition is false.
    • Use elif to check multiple conditions.
    • Indentation is mandatory in Python.
