📂 Python File Handling

Lesson Progress

File Handling allows us to create, read, write and modify files using Python.

What is File Handling?

File Handling is used to store data permanently in files. Python provides built-in functions to create, read, write and append files.

Example


        file = open("demo.txt", "w")
        
        file.write("Welcome to AIPath")
        
        file.close()

Output


        A file named demo.txt is created successfully.

💡 Key Points

  • open() is used to open a file.
  • "r" is Read Mode.
  • "w" is Write Mode.
  • Always close the file after use.
