⚙️ JavaScript Functions

Functions are reusable blocks of code that perform a specific task. They help make programs cleaner, organized, and easier to maintain.

Lesson 6 of 8

⚙️ What is a Function?

A function is a reusable block of code that performs a specific task. Instead of writing the same code again and again, we can place it inside a function and call it whenever needed.

💡 Real Life Example

Think of a TV Remote 📺. Every button performs a different task. Pressing the Volume button always increases or decreases the volume. Similarly, a function performs the same task whenever it is called.

📝 Function Declaration

💻 Syntax


        function greet(){
        
            console.log("Welcome to AIPath!");
        
        }
        
        greet();

🖥 Output

        Welcome to AIPath!

📥 Parameters & Arguments

Parameters are variables written inside the function definition. Arguments are the actual values passed when calling the function.

💻 Example


        function greet(name){
        
            console.log("Welcome " + name);
        
        }
        
        greet("Pooja");

🖥 Output

        Welcome Pooja

📤 Return Statement

The return statement sends a value back to the place where the function was called. 


        function add(a, b){
        
            return a + b;
        
        }
        
        let result = add(10, 20);
        
        console.log(result);

🖥 Output

        30

🎯 Common Interview Question

Q. What is the difference between a Parameter and an Argument?

Answer: Parameters are variables defined in the function declaration, while arguments are the actual values passed when the function is called.

🧠 Did You Know?

Functions are one of the most important concepts in JavaScript. Modern frameworks like React, Angular, and Node.js rely heavily on functions.

💡 Key Points

  • Functions help avoid repeating code.
  • A function can accept parameters.
  • Arguments are values passed to a function.
  • The return statement sends a value back.
  • Functions make code reusable and easier to maintain.
