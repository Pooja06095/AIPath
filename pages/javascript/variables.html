📦 JavaScript Variables & Data Types

Variables are used to store data in JavaScript. Data Types define what kind of value a variable can hold.

Lesson 2 of 8

📦 What is a Variable?

A variable is a container that stores data. It allows you to save information and use it later in your program.

💡 Real Life Example

Think of a variable like a labeled box. You can store different items inside it and use them whenever you need.

📝 Declaring Variables

JavaScript provides three ways to declare variables.

  • let → Modern and recommended.
  • const → Value cannot be changed.
  • var → Old method (avoid using in new projects).

📚 JavaScript Data Types

🔤 String
🔢 Number
✅ Boolean
📦 Object
📋 Array
❌ Null

💻 Example


        let studentName = "Pooja";
        let cgpa = 7.4;
        const college = "BIST";
        let isPlaced = false;
        
        console.log(studentName);
        console.log(cgpa);
        console.log(college);
        console.log(isPlaced);

🖥 Output

        Pooja
        7.4
        BIST
        false

📌 Variable Naming Rules

  • ✅ Variable names can contain letters, numbers, _ and $.
  • ✅ Variable names cannot start with a number.
  • ✅ JavaScript is case-sensitive.
  • ✅ Use meaningful names like studentName instead of x.

🧠 Did You Know?

💡 Key Points

  • Variables store data.
  • let is the recommended way to declare variables.
  • const is used for values that should not change.
  • var is an older way to declare variables.
  • Always use meaningful variable names.
