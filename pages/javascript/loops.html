🔁 JavaScript Loops

Loops help us execute the same block of code multiple times without writing it again and again.

Lesson 5 of 8

🔁 What are Loops?

A loop repeats a block of code until a condition becomes false.

💡 Real Life Example

Imagine a teacher taking attendance. Instead of calling one student manually every time, the teacher repeats the same process for every student in the class.

📚 Types of Loops

🔁 for Loop
🔄 while Loop
🔂 do...while Loop
🎯 for...of Loop

🔁 for Loop

The for loop is used when you know how many times you want to repeat a block of code.

💻 Example


        for(let i = 1; i <= 5; i++){
        
            console.log(i);
        
        }

🖥 Output

        1
        2
        3
        4
        5

🎬 Loop Visualization

        Iteration 1 → i = 1 ✅
        
        Iteration 2 → i = 2 ✅
        
        Iteration 3 → i = 3 ✅
        
        Iteration 4 → i = 4 ✅
        
        Iteration 5 → i = 5 ✅
        
        Condition becomes FALSE ❌
        
        Loop Stops

🔄 while Loop

The while loop runs as long as the condition is true. 


        let i = 1;
        
        while(i <= 3){
        
            console.log(i);
        
            i++;
        
        }

🔂 do...while Loop

The do...while loop executes the code at least once, even if the condition is false. 


        let i = 1;
        
        do{
        
            console.log(i);
        
            i++;
        
        }while(i <= 3);

🎯 Common Interview Question

Q. What is the difference between while and do...while loop?

Answer: A while loop checks the condition first, whereas a do...while loop executes the code once before checking the condition.

🧠 Did You Know?

Infinite loops can freeze your browser or application. Always make sure the loop condition eventually becomes false.

💡 Key Points

  • Loops repeat a block of code.
  • for is best when the number of iterations is known.
  • while is used when the number of iterations is unknown.
  • do...while always executes at least once.
  • Avoid infinite loops.
