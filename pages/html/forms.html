📝 What are HTML Forms?

HTML Forms are used to collect information from users. They allow users to enter data such as names, emails, passwords, phone numbers, and more.

📚 Common Form Elements

  • <form> - Creates a form
  • <input> - Takes user input
  • <label> - Defines a label
  • <textarea> - Multi-line input
  • <select> - Drop-down list
  • <button> - Creates a button

💻 Example


    

    <form>
    
        <label>Name:</label>
        <input type="text">
    
        <br><br>
    
        <label>Email:</label>
        <input type="email">
    
        <br><br>
    
        <button>Submit</button>
    
    </form>

💡 Key Points

  • HTML Forms are used to collect user information.
  • The <form> tag creates a form.
  • The <input> tag is used to take user input.
  • Forms can include text fields, email fields, passwords, checkboxes, radio buttons, and buttons.
