📊 What are HTML Tables?

HTML Tables are used to display data in rows and columns. They are useful for showing structured information like marks, prices, schedules, and reports.

📚 Common Table Tags

  • <table> - Creates a table
  • <tr> - Creates a table row
  • <th> - Creates a table heading
  • <td> - Creates a table data cell
  • <caption> - Adds a table title

💻 Example


            <table border="1">
            
                <tr>
                    <th>Name</th>
                    <th>Age</th>
                </tr>
            
                <tr>
                    <td>Pooja</td>
                    <td>20</td>
                </tr>
            
                <tr>
                    <td>Rahul</td>
                    <td>21</td>
                </tr>
            
            </table>
Visit Google

🖥 Output

Name Age
Pooja 20
Rahul 21

💡 Key Points

  • HTML Tables display data in rows and columns.
  • <table> creates the table.
  • <tr> creates a row.
  • <th> creates a heading cell.
  • <td> creates a data cell.
