🏗 What is Semantic HTML?

Semantic HTML uses meaningful tags that describe the purpose of the content. These tags make webpages easier to understand for both browsers and developers.

📚 Common Semantic Tags

  • <header> - Website Header
  • <nav> - Navigation Menu
  • <main> - Main Content
  • <section> - A Section of Content
  • <article> - Independent Article
  • <footer> - Website Footer

💻 Example


    <header>
        <h1>AIPath</h1>
    </header>
    
    <nav>
        Home | Courses | Contact
    </nav>
    
    <main>
        <section>
            Welcome to AIPath!
        </section>
    </main>
    
    <footer>
        © 2026 AIPath
    </footer>
Visit Google

🖥 Output

AIPath

Home | Courses | Contact

Welcome to AIPath!

© 2026 AIPath

💡 Key Points

  • Semantic HTML uses meaningful tags.
  • It improves readability and code structure.
  • It helps Search Engines understand your website.
  • It improves accessibility for screen readers.
