🏷 HTML Tags

Course Progress

HTML Tags are the building blocks of every webpage. They tell the browser how to display content.

🏷 What are HTML Tags?

HTML Tags are special keywords enclosed in angle brackets (< >). They define the structure and content of a webpage.

📚 Common HTML Tags

  • <h1> - Main Heading
  • <p> - Paragraph
  • <a> - Hyperlink
  • <img> - Image
  • <ul> - Unordered List
  • <ol> - Ordered List

💻 Example


    
Welcome to AIPath

    
    
This is my first webpage.

    
    Visit Google

🖥 Output

Welcome to AIPath

This is my first webpage.

Visit Google

💡 Key Points

  • HTML Tags define the structure of a webpage.
  • Most tags have opening and closing tags.
  • Tags are written inside angle brackets (< >).
  • Each tag has a specific purpose.
