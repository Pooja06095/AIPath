💻 Installing Git

Learn how to download, install and verify Git on your computer.

Lesson 2 of 8

📌 Why Install Git?

Before using Git commands, Git must be installed on your computer. Once installed, you can track changes, create repositories, and collaborate with other developers.

🌐 Download Git

Download Git from the official website.

https://git-scm.com/downloads

🖥 Installation Steps

  1. Open the Git official website.
  2. Download Git for Windows.
  3. Run the installer.
  4. Keep the default settings.
  5. Click Next until Install appears.
  6. Click Install.
  7. Click Finish.

📸 Installation Preview

This is the Git installation window. Simply keep the default options and click Next until Finish.

💻 Verify Installation

Open Command Prompt or Git Bash and run:

git --version

If Git is installed correctly, you'll see something like:

git version 2.51.0

⚠️ Common Installation Errors

  • Downloading Git from unofficial websites.
  • Skipping the installation process before using Git commands.
  • Not restarting Command Prompt after installation.
  • Typing the command incorrectly (git --version).

💡 Pro Tip

Always download Git from the official website (https://git-scm.com). During installation, the default settings are suitable for almost all beginners.

🧠 Common Interview Question

Q. How can you verify that Git is installed successfully?

Answer: Open Command Prompt or Git Bash and run:

git --version

💡 Did You Know?

Git is free and open-source software. It was created by Linus Torvalds in 2005, the creator of the Linux operating system.

📋 Key Points

  • Git must be installed before using Git commands.
  • Download Git only from the official website.
  • Use default installation settings.
  • Verify installation using git --version.

📝 Practice

  • Visit the official Git website.
  • Download Git for your operating system.
