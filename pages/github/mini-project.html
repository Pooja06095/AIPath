🚀 Mini Project - Upload Your First Website

Apply everything you've learned by creating a small website and uploading it to GitHub.

Lesson 8 of 8

🎯 Project Objective

In this project you will create a simple website and publish it on GitHub using Git commands.

📂 Project Structure

        
        My-First-Website
        
        │
        
        ├── index.html
        
        ├── style.css
        
        └── README.md

🌐 Step 1 - Create index.html

        
        
Hello, I'm Pooja

        
        
Welcome to my first website.

🎨 Step 2 - Create style.css

        
        body{
        
        font-family:Arial;
        
        text-align:center;
        
        padding:50px;
        
        }
        
        h1{
        
        color:#2563eb;
        
        }

👀 Preview

Open index.html in your browser. You should see your first website.

⚙ Initialize Git

git init

📁 Add Files

git add .

💾 Commit Changes

git commit -m "First Website"

🌍 Create Repository

Open GitHub. Click New Repository. Repository Name:

my-first-website

🔗 Connect Local Project

git remote add origin YOUR_REPOSITORY_URL

🚀 Upload Project

git push -u origin main

✅ Verify

Refresh your GitHub repository. Your project files should now appear online. 🎉 Congratulations!
